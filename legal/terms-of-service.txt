1. THE SERVICE

PRismAI-Reviewer is an automated code review assistant that integrates with GitHub as a GitHub App.
When installed on a repository, PRismAI-Reviewer analyses pull request diffs using large language
models and provides automated feedback as GitHub comments.

The Service includes features that vary by subscription plan, including but not limited
to: bug detection, design analysis, performance analysis, fix suggestions, risk scoring,
reviewer recommendations, and retrieval-augmented contextual review.
